## Description
Set guild's anti channel delete limit or disable it.

## Usage
!!!primary Set limit
`-antichanneld <1/2/3/4/...>`
--![antichannel delete value](../../img/Commands/AutoMod/antichanneld.png)--
!!!

!!!
